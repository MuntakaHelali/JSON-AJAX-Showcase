# JSON-AJAX-Showcase
Sipmle webpage to showcase the ability of using JSON and AJAX to fetch animal data and display it from another source.
Copy/Click this link https://muntakahelali.github.io/JSON-AJAX-Showcase/ if the webpage from the website section fails to work.
<br>
<br>
The webpage contains a simple grey background with a modal on top where the information of the animals are fetched using AJAX and JSON formatting.
